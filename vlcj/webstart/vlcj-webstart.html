vlcj

Java bindings for the vlc media player

Information

The vlcj project provides Java bindings for the excellent VLC media player from VideoLAN.

The bindings can be used to build media player client and server software using Java - everything from simply playing local media files to a full-blown video-on-demand streaming server is possible.

This page allows you to download and execute the test media player application that is included with vlcj. Please check the demo requirements and click the Launch button below to try it out!

The libvlc plugins download is approximately 14Mb so it may take some time for the application to start.

Demo Requirements

The vlcj bindings have a minimum requirement of Java 5. The demo video player requires Java 6.

The native libraries for vlc that are included in this WebStart download are those for 64-bit Linux. This demo has been confirmed to work without problems on Ubuntu 10.10.

If you are not using a 64-bit Linux operating system then this WebStart demo simply will not work. This restriction does not apply to the stand-alone test player.

Since native libraries are used, the application requires all security permissions. You must accept any security warning dialogs if you want to see the demo.
